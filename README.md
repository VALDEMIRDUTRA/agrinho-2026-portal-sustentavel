# Portal AgroSustentável

> Projeto desenvolvido para o Concurso Agrinho 2026 - Categoria Programação.
> Tema Oficial: "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente".

---

## Sobre o Projeto

Este portal educativo foi criado com o objetivo de demonstrar como é possível manter o agronegócio forte e, ao mesmo tempo, preservar a natureza. A plataforma apresenta técnicas de sustentabilidade agrícola de forma simples e desafia o usuário a testar o que aprendeu por meio de um sistema interativo.

Acreditamos que a união entre educação e tecnologia é o melhor caminho para conscientizar a sociedade sobre a importância da preservação ambiental no campo.

---

## Funcionalidades da Aplicação

* **Área Informativa:** Apresentação clara de conceitos como Rotação de Culturas e Plantio Direto.
* **Quiz Interativo:** Sistema de perguntas e respostas integrado ao conteúdo do portal.
* **Feedback em Tempo Real:** Validação imediata da resposta do usuário, com mudança visual de cores e explicação técnica do acerto ou erro.
* **Design Responsivo:** Interface limpa e adaptável, garantindo uma boa navegação tanto em computadores quanto em dispositivos móveis.

---

## Tecnologias Utilizadas

Para cumprir rigorosamente as regras do edital do Concurso Agrinho, este projeto foi desenvolvido do zero, utilizando apenas tecnologias nativas da web (Vanilla), sem o uso de frameworks externos.

| Tecnologia | Aplicação no Projeto |
| :--- | :--- |
| **HTML5** | Estruturação semântica, organização do conteúdo e textos (`index.html`). |
| **CSS3** | Estilização visual, paleta de cores, layout e responsividade (`style.css`). |
| **JavaScript** | Lógica de programação do quiz e interatividade com o usuário (`script.js`). |
